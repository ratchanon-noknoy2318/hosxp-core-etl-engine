## Data Investigation Workflow

This project followed an iterative, data-driven approach to understanding clinical data:
- Listened to frontline staff to understand clinical concepts (e.g., DTX usage and timing)
- Traced database schemas, tables, and fields to identify relevant data sources
- Used SQL and LLM tools to generate hypotheses and explore possible interpretations of the data
- Validated findings with domain experts to ensure alignment with real-world clinical workflowskflows.
